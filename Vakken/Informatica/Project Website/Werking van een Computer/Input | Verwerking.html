Input/Invoer

De input van een computer is alles wat je aan de computer geeft.Bijvoorbeeld: Een Toetsenbord, de toetsen die je dan indruk is de invoer. Dit hoeft niet alleen tekst te zijn, maar het kan ook bijvoorbeeld een microfoon, muis of een scanner. Dan gaat de informatie door naar de verwerking of ook wel de processing

Verwerking/Processing

Nadat de computer alle inputs heeft binnen gekregen. Al deze inputs moeten verwerkt worden tot informatie. Dit doet de CPU, het brein van een computer. De CPU kan eigenlijk vanalles doen, zoals het bereken van een som, maar ook dus het verwerken van inputs tot informatie. De Ram slaat allerlei dingen zoals informatie die de CPU aan het gebruiken is tijdens het bereken. Maar ook alle inputs, wanneer deze tot informatie moeten worden vermaakt.
